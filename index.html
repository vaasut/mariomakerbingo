Mario Maker Endless Bingo!

Suggested Rules

Here's a little bit of guidance for how to play. Feel free to include or exclude any of these rules.

1. Skips allowed.
2. If you lose all your lives, you can start a new run with the current board. Or you can start fresh with a new board if you want it to be extra challenging.
3. To check if a level is a water level (snow level, castle level, etc) check if the overworld displays that theme.
